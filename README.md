# Emotion Recognition from Speech using Deep Learning

## Overview

This project is a Speech Emotion Recognition (SER) system that predicts human emotions from speech audio using Deep Learning.

## Features

- Speech Emotion Recognition
- Audio Preprocessing
- Feature Extraction
- CNN + BiLSTM Model
- Streamlit Web Application
- Confidence Score Prediction

## Tech Stack

- Python
- TensorFlow
- Librosa
- Streamlit
- Scikit-learn
- NumPy
- Pandas

## Dataset

RAVDESS Dataset
